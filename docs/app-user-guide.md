# App User Guide

This guide describes how to use the current meditation-community app.

## Main navigation

The app uses four bottom tabs:

- `Home`
- `Library`
- `Growth`
- `Settings`

`SOS` is not a tab. It is available from the Home flow and other high-stress entry points.

## Home

Home is the primary starting point.

What you can do there:

- start a suggested session
- use a quick emotional or energy check-in
- open `SOS`
- continue a ritual or recent flow
- view a quiet presence summary

Home is designed to show only a small number of actions at once.

## SOS

Use `SOS` when you want the fastest possible calming flow.

Typical path:

1. Open `Home`
2. Tap `SOS`
3. Start the active calming flow
4. Exit or return home whenever needed

The flow is intentionally minimal and always keeps an exit path visible.

## Library

`Library` helps you browse sessions by:

- need
- context
- duration
- intensity
- transition mode
- pack type

You can tap a session to open the session player.

## Session player

The session player is where a meditation session runs.

You can expect:

- timer and bell cues
- pause and resume controls
- a gentle re-entry screen after completion
- fallback behavior when offline or when sensor permissions are missing

## Growth

`Growth` shows progress without streak pressure.

It focuses on:

- total sessions
- total duration
- reflection trends
- optional follow-up actions

It does not center streaks or leaderboard-style metrics.

## Presence

`Presence` shows anonymous aggregate community presence.

It does not show:

- chat
- feed
- comments
- profile identity

You can join or leave a quiet presence block without creating a social graph.

## Permissions

Some features require a pre-permission explanation screen before the system prompt.

This is used for things like microphone or health access.

## Settings

`Settings` contains runtime and accessibility toggles, including haptic preference.

## Offline and privacy behavior

The app is designed to keep core flows usable when:

- network is unavailable
- sensor permissions are missing
- some content is cached locally only

Privacy-sensitive features are kept explicit and permission-gated.

## If you want a quick start

1. Open `Home`
2. Use the top suggested CTA
3. If you are overwhelmed, tap `SOS`
4. If you want a shorter session, open `Library`
5. After a session, use the re-entry or reflection actions
